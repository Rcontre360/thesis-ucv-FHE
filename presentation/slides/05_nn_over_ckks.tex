\begin{frame}{Inferencia Privada}
    La inferencia privada permite que un cliente utilice un modelo de aprendizaje automático alojado en un servidor remoto sin revelar sus datos de entrada ni el resultado de la predicción. Mediante el uso de cifrado homomórfico, el servidor puede ejecutar el modelo directamente sobre los datos cifrados, garantizando que la información sensible nunca sea expuesta durante el proceso de cómputo.

\end{frame}

\begin{frame}{Multiplicación Matriz-Vector y Diagonalización}
    Optimización para minimizar rotaciones homomórficas (Halevi-Shoup).

    \begin{itemize}
        \item \textbf{Diagonales} Extracción de las diagonales de la matriz de pesos como vectores independientes.
        \item \textbf{Rotacion:} Rotación cíclica del vector de entrada y producto de Hadamard con cada diagonal.
        \item \textbf{Results addition:} Acumulación de los productos parciales.
    \end{itemize}

\begin{frame}{Aproximación de Funciones de Activación}
    El esquema CKKS solo soporta sumas y multiplicaciones polinomiales.

    \begin{itemize}
        \item \textbf{Problema} Funciones no lineales (ReLU, Sigmoid) no pueden evaluarse directamente.
        \item \textbf{Solucion:} Aproximar las funciones no lineales usando polinomios en una rango dado.
        \begin{itemize}
            \item \textbf{Series de Taylor:} Útiles para funciones suaves (Sigmoid, Tanh).
            \[ \text{Sigmoid}(x) \approx \frac{1}{2} + \frac{1}{4}x - \frac{1}{48}x^3 + \dots \]
            \item \textbf{Funcion cuadrada:} el uso de la función cuadrada $f(x) = x^2$ como una alternativa a ReLU de bajo coste computacional.
                \[ \text{ReLU}(x) \approx x^2 \]
        \end{itemize}
    \end{itemize}
\end{frame}

\begin{frame}{Convoluciones Homomórficas e Im2Col}
    Transformación de vecindades espaciales en estructuras lineales.
    
    \begin{itemize}
        \item \textbf{Mecánica:} El parche de la imagen y el kernel se aplanan en vectores.
        \item \textbf{Propiedad:} La convolución se reduce a un \textbf{producto punto} entre dos vectores.
        \item \textbf{Simplicidad} Permite resolver la capa mediante una única multiplicación de matrices (Diagonalización).
    \end{itemize}

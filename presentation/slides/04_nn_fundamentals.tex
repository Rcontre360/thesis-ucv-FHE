\section{Fundamentos de Redes Neuronales}
\SectionFrame{Fundamentos de Redes Neuronales}

\begin{frame}{Redes Neuronales y Redes Superficiales}
    Las \textbf{Redes Neuronales Artificiales (RNA)} son modelos computacionales diseñados para aprender patrones y relaciones complejas a partir de datos. Las \textbf{Redes Superficiales} representan la arquitectura más sencilla, caracterizada por tener una única capa oculta entre la entrada y la salida:
    
    \begin{equation*}
        y = a_2 \left[ \beta_2 + \Omega_2 (a_1 [\beta_1 + \Omega_1 x]) \right]
    \end{equation*}

\end{frame}

\begin{frame}{Componentes de la Neurona}
    La unidad fundamental de procesamiento en una red es la neurona, cuya operación $h = a[\beta + \Omega x]$ se basa en los siguientes elementos:
    \begin{itemize}
        \item \textbf{Pesos ($\Omega$):} Parámetros que determinan la importancia o fuerza de cada señal de entrada.
        \item \textbf{Sesgo ($\beta$):} Valor independiente que permite desplazar la función de activación para ajustar el umbral de respuesta.
        \item \textbf{Función de Activación ($a$):} Función (usualmente no lineal) que transforma la suma ponderada en la salida final de la neurona.
    \end{itemize}
\end{frame}

\begin{frame}{Redes Neuronales Profundas}
    Las \textbf{Redes Neuronales Profundas (DNN)} se caracterizan por poseer dos o más capas ocultas. Esta profundidad permite al modelo aprender representaciones jerárquicas y abstractas de los datos, resolviendo problemas de mayor complejidad y dimensionalidad.

\begin{frame}{Redes Neuronales Convolucionales (CNN)}
    Especializadas en datos con estructura espacial, las CNN utilizan la operación de convolución para extraer características jerárquicas mediante los siguientes parámetros clave:
    \begin{itemize}
        \item \textbf{Filtros:} Matrices que extraen patrones locales.
        \item \textbf{Stride:} Define el desplazamiento del filtro sobre la entrada.
        \item \textbf{Padding:} Borde añadido para controlar las dimensiones de salida.
        \item \textbf{Canales:} Profundidad de los datos (ej. RGB o mapas de características).
    \end{itemize}
